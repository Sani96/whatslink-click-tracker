== Frequently Asked Questions ==

= Does it track all WhatsApp links? =
Yes! It works with both `https://wa.me/` and `https://api.whatsapp.com/send` links.

= Is it compatible with WooCommerce? =
Absolutely. You can track clicks on product pages too.

= Can I export logs? =
Only in the Pro version, where you get CSV export and email reports.

= Will it slow down my site? =
Not at all. The plugin is optimized for performance and loads asynchronously.
